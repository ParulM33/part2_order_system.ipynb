## Assignment Overview

This project is a Python application that demonstrates:

- File input and output operations

- API integration using HTTP requests

- Robust exception handling

- Error logging with timestamps

 

The program fetches real product data from the public **DummyJSON API**, processes it, saves information to files, and handles failure scenarios gracefully.

 

This assignment was implemented as a **Google Colab / Jupyter Notebook** and is suitable for execution in environments supporting Python 3.

 

---

 

## Tools & Technologies Used

- Python 3

- Google Colab / Jupyter Notebook

- `requests` library for API interaction

- File handling with UTF‑8 encoding

- Exception handling (`try` / `except` / `finally`)

- `datetime` module for logging timestamps
